# Premium Music Player

A sleek, dark-themed music player with glassmorphism effects and Spotify-inspired interface.

## Features
- Premium dark glassmorphism UI
- Floating sparkle effects
- Spotify-like interface
- Fully functional player controls
- Playlist management
- Responsive design
- Keyboard shortcuts

## Setup
1. Clone the repository
2. Add your MP3 files in the `songs/` directory
3. Add corresponding thumbnails or use Unsplash URLs
4. Deploy to GitHub Pages or Render

## Keyboard Shortcuts
- Space: Play/Pause
- Arrow Right: Next Song
- Arrow Left: Previous Song
- M: Mute/Unmute
